## Foundational Rule — No Shadow Documentation

This repository shall not duplicate, summarize, rewrite, or replace
official Gaggiuino installation steps.

All installation procedures must be followed directly from the
Official Gaggiuino Documentation (verbatim copy stored in this repo).

This repository may:
- Route to the correct official section
- Provide safety clarification
- Define prerequisites
- Define rollback points
- Provide risk classification
- Clarify terminology

This repository may NOT:
- Rewrite step sequences
- Shorten installation instructions
- Reorder official procedures
- Omit official steps
- Act as a substitute reference

The Official Documentation remains uncontested and primary at all times.
