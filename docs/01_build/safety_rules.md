# Safety Rules

These rules are mandatory.

## Electrical Safety

- Never work on the machine while plugged in.
- Lid ON when plugged in.
- Lid OFF only when unplugged.
- Confirm unplug status before any internal access.

## Live Power

If a step requires live power:
- Explicit confirmation required.
- Risk explained before proceeding.
- No improvisation allowed.

## Philosophy

- Prefer reversible steps.
- Define rollback points.
- Overbuild > shortcut.
- If uncertain: stop and ask.
